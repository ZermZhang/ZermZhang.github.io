使用Gridea和Github创建个人博客

2020-06-27 2 min read # Github # Gridea

1. github的相关配置

  1. 创建自己的git账号
  2. 如下操作创建personal access token

    右上角头像 -> setting -> Developer Setting -> personal access tokens -> Generate new token

    1. 创建token的时候只需要勾选repo权限即可
  3. 创建博客需要用到的仓库
    1. 需要仓库权限为public
    2. 仓库名最好为用户名.github.io，方便Github将此仓库设置成Githu Pages可访问的类型

2. Gridea的相关配置

  1. 基础配置
  2. 保存后会自动生成相关模板和一个初始化的页面（如下）
  3. 点击同步后Gridea便会将相关内容同步到github上
  4. 通过在Gridea中配置的域名即可访问你的博客了

3. 针对Gridea写作的一些说明

  1. Gridea的文件结构

    • config：基本配置文件
    • images：图片存储位置
    • output：经过处理的博客源文件[也就是会同步到github上的文件夹]
    • post-images：关联的图片文件
    • posts：文章存储位置
    • static：404页面
    • themes：主题

  2. 使用其他编辑器编辑文章内容

    • 可以通过Gridea创建了文章之后使用其他的编辑器打开文章进行编辑
    • 也可以使用其他编辑器编辑之后将.md文件移动到/posts文件加下面
      • 注意：文章标题格式需要使用如下格式
      ---
title: 'chang-jian-shell-ming-ling'
date: 2018-12-12 00:00:00
tags: [Gridea]
published: true
hideInList: true
feature:

---
